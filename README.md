# Detection Engineering Lab

## Project Overview

This project demonstrates detection engineering skills using Windows Event Logs.

The objective was to analyze Windows Security and System logs, identify important security events, create detection rules, and document investigation procedures similar to a SOC analyst workflow.

---

## Technologies Used

- Windows 11
- Event Viewer
- Windows Security Logs
- MITRE ATT&CK Framework
- GitHub
