# Detection Engineering Lab Report

## Objective

To build and document Windows security detections using native Windows Event Logs.

## Methodology

1. Collected Windows Security Events
2. Investigated Security Event IDs
3. Developed Detection Rules
4. Mapped Events to MITRE ATT&CK
5. Documented Investigation Procedures

## Results

Successfully developed 9 detection rules covering:

- Authentication Monitoring
- Process Monitoring
- Service Installation Monitoring
- Audit Policy Monitoring
- Persistence Detection

## Conclusion

The lab demonstrates practical detection engineering and SOC analyst skills using real Windows logs.
