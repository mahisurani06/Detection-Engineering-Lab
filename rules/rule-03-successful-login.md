# Detection Rule 03 – Successful Login Monitoring

## Objective

Monitor successful logins and identify unusual account access.

## Event ID

4624 – Successful Logon

## Detection Logic

Alert when:

* Privileged account logs in
* Login occurs outside business hours
* Login originates from unusual IP address
* Login type is Remote Interactive (10) or Network (3)

## Severity

Medium

## MITRE ATT&CK

TA0001 – Initial Access

## Observed Event

Account:
SYSTEM

Logon Type:
5 (Service)

Process:
C:\Windows\System32\services.exe

Source IP:
N/A

## Investigation Steps

1. Identify logged-in account.
2. Verify logon type.
3. Check source IP.
4. Review related authentication events.
5. Determine if account activity is expected.

## Analyst Conclusion

Service account login generated by Windows.

Legitimate operating system activity.

Risk Level: Low
